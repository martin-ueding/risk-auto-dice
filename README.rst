risk-auto-dice
##############

Does the dice rolling in the classic game “Risk”. See
http://martin-ueding.de/projects/risk-auto-dice/.

Installation
============

::

    make
    sudo make install

Dependencies
============

- Python 2 (``python``)
- ``gettext``
